This is a PyTorch implementation of SEQ2SEQ-HMF for HFMD prediction.

Under the ‘data’ directory, the files hfmd_num1, weather, weekend, adj1, and jpdadj contain the organized data related to Japanese HFMD. The data sources are the National Institute of Infectious Diseases of Japan and the Japan Meteorological Agency. The files cn and cnadj come from the China NCDC.
